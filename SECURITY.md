# Security

This repository is intended for public workflow files, prompts, and agent instructions.

Please do not open public issues containing secrets, credentials, private URLs, or sensitive configuration. If you notice sensitive information in the repository, contact the maintainer privately so it can be removed and rotated.

Before contributing or publishing updates, review changes for:

- API keys and tokens.
- Private service URLs.
- Internal repository names.
- Client or customer information.
- Personal filesystem paths.
- Local logs, caches, or generated tool state.
